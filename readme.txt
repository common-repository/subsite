== Description ==

NEW FOR 2024! 

Create a "site within a site" without using WordPress networks or separate WordPress installations!

= Features: = 

For a defined set of pages on your site: 

* Change the logo to a different logo
* Change which menus appear
* (Pro version only) Display the subsite at its own subdomain (e.g. subsite.yourdomain.com)
* (Pro version only) Apply custom CSS to specific subsites
* (Pro version only) Allow users to search only within a particular subsite
* (Pro version only) Allow content managers to attach a page to a subsite from the editing screen
